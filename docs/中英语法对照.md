# 中英语法对照

## 保留关键字

| 中文关键字 | 英文关键字 |
| :-----: | :-----: |
| 导入 | import |
| 从 | from |
| 返回 | return |
| 跳出 | break |
| 继续 | continue |

## 内置函数

| 中文函数 | 英文函数 |
| :-----: | :-----: |
| 打印 | print |
| 输入 | input |
| 范围 | range |

## 内置属性

| 中文属性 | 英文属性 |
| :-----: | :-----: |
| 替换 | replace |

## 装饰器

| 中文装饰器 | 英文装饰器 |
| :-----: | :-----: |
| 静态方法 | staticmethod |
| 类方法 | classmethod |
