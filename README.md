## 原理

使用正则将注释,引号内容给替换成占位符(会自动根据代码生成唯一占位符)

再嵌套替换中括大括号号为占位符(这样做就不会因为[ "]" ]而提前结束匹配)

然后就是硬核正则替换函数为Python函数

最后将占位符还原

现在实现了自己编译自己
